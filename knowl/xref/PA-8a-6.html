Preview Activity 8.7.1.
Consider the initial value problem given by
\begin{equation*} y' = y, \ y(0) = 1\text{.} \end{equation*}
🔗
🔗
(a)
To begin, let’s assume that \(y\) is a function with a power series representation that is centered at \(a = 0\text{,}\) so we can write
\begin{equation} y = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots + c_n x^n + \cdots\text{.}\tag{8.7.1} \end{equation}
Use the The Power Series Differentiation and Integration Theorem to differentiate the right side of Equation (8.7.1). Thus, express \(y'\) as a power series; show at least the first four nonzero terms of that series.
🔗
🔗
(b)
An important fact about polynomials is that the only way two polynomials can be equal to one another is if they have the exact same coefficients for each power of \(x\text{.}\) For example, if we know that \(a_0 + a_1 x + a_2 x^2 = 3 - 2x + 5x^2\text{,}\) then it follows that \(a_0 = 3\text{,}\) \(a_1 = -2\text{,}\) and \(a_2 = 5\text{.}\)
🔗
This fact about polynomials extends to power series: the only way that two power series can be equal is for them to have identical coefficients. In (a), you found that \(y' = c_1 + 2c_2 x + 3c_3 x^2 + 4c_4 x^3 + \cdots + nc_n x^{n-1} + \cdots\text{.}\) Using the power series representation of \(y\) found in Equation (8.7.1), together with the condition from the initial value problem that \(y' = y\text{,}\) it follows that
\begin{equation} c_1 + 2c_2 x + 3c_3 x^2 + \cdots = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + \cdots\tag{8.7.2} \end{equation}
What does Equation (8.7.2) imply about the relationship between \(c_0\) and \(c_1\text{?}\)
🔗
🔗
(c)
Note that the initial value problem statement tells us that \(y(0) = 1\text{.}\) Using Equation (8.7.1) determine the value of \(c_0\text{,}\) and hence find the value of \(c_1\text{.}\)
🔗
🔗
(d)
Now return to Equation (8.7.2). What can we conclude about the relationship between \(c_1\) and \(c_2\) based on the “\(x\)” terms on each side of the equation? Using this relationship and the value you found for \(c_1\) in (c), determine the numerical value of \(c_2\text{.}\)
🔗
🔗
(e)
Finally, reason similarly to find \(c_3\text{,}\) \(c_4\text{,}\) and \(c_5\) by equating the coefficients of \(x^2\text{,}\) \(x^3\text{,}\) and \(x^4\text{,}\) respectively, in Equation (8.7.2). What do you find? Why is this result not surprising?
