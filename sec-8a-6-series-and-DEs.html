Section 8.6 Using Series to Solve Differential Equations

Motivating Questions

  • How can power series be used to find solutions to familiar differential equations?

  • In what ways do series solutions to differential equations align with solutions we can find without using series?

  • What role do series solutions play in differential equations that can't be solved by other approaches?

In our work in Chapter 7, we learned that the natural exponential function, \(e^x\text{,}\) plays a key role in the solution of several important types of differential equations. In our more recent studies in Chapter 8, we've learned that \(e^x\) can be represented as a Taylor series in the form

\begin{equation*} e^x = c_0 + c_1 x + c_2 x^2 + \cdots + c_n x^n + \cdots = \sum_{k=0}^{\infty} c_k x^k\text{.} \end{equation*}

For \(f(x) = e^x\text{,}\) we know \(f^{(k)}(x) = e^x\) for every natural number \(k\text{,}\) from which it follows that \(f^{(k)}(0) = 1\) for every \(k\text{.}\) This fact enabled us to determine the values of each coefficient \(c_k\) for the Taylor series expansion of \(f(x) = e^x\text{,}\) and hence establish that

\begin{equation*} e^x = 1 + x + \frac{1}{2!}x^2 + \cdots + \frac{1}{n!}x^n + \cdots = \sum_{k=0}^{\infty} \frac{1}{k!} x^k\text{,} \end{equation*}

which is valid for every real number \(x\text{.}\)

In Preview Activity 8.6.1, we explore a different approach to finding the series representation of \(f(x) = e^x\text{.}\)

Subsection 8.6.1 Using power series to find solutions to initial value problems

In Preview Activity 8.6.1, we investigated the initial value problem given by

\begin{equation*} y' = y, \ y(0) = 1\text{.} \end{equation*}

By assuming that the solution function \(y\) had a power series representation, we were able to write that

\begin{equation*} y = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots + c_n x^n + \cdots\text{,} \end{equation*}

and by finding \(y'\) and setting \(y' = y\text{,}\) see that the equation

\begin{equation*} c_1 + 2c_2 x + 3c_3 x^2 + 4c_4 x^3 + \cdots = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots \end{equation*}

must hold. This led us to conclude that \(c_1 = c_0\text{,}\) \(2c_2 = c_1\text{,}\) \(3c_3 = c_2\text{,}\) and \(4c_4 = c_3\) by equating like coefficients. Since \(y(0) = 1\) is given in the IVP and \(y(0) = c_0\) from the series representation, it follows that \(c_0 = 1\text{,}\) and from there that \(c_1 = 1\text{,}\) \(c_2 = \frac{1}{2}\text{,}\) \(c_3 = \frac{1}{3 \cdot 2}\text{,}\) and \(c_4 = \frac{1}{4 \cdot 3 \cdot 2}\text{.}\)

This of course suggests the pattern that \(c_n = \frac{1}{n!}\text{,}\) and hence that the solution to the initial value problem is

\begin{equation*} y = 1 + x + \frac{1}{2!}x^2 + \frac{1}{3!}x^3 + \cdots + \frac{1}{n!}x^n + \cdots\text{.} \end{equation*}

Remember, we found this representation of \(y\) by solving the IVP

\begin{equation*} y' = y, \ y(0) = 1\text{,} \end{equation*}

which states that “\(y\) is a function whose derivative is itself and that satisfies \(y(0) = 1\)”: this is precisely the function \(y = e^x\text{,}\) so our approach here has established the power series representation of \(f(x) = e^x\) from a different line of reasoning than Taylor series. Indeed, we've found that

\begin{equation*} e^x = 1 + x + \frac{1}{2!}x^2 + \frac{1}{3!}x^3 + \cdots + \frac{1}{n!}x^n + \cdots\text{.} \end{equation*}

This approach can be used to solve a select class of other initial value problems. The key idea is to let \(y = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots + c_n x^n + \cdots\text{,}\) find the corresponding expression for \(y'\) (and possibly \(y''\) and other higher derivatives if the differential equation has higher order), and then subsitute these series expresssions into the differential equation to determine relationships that must hold among the unknown coefficients, \(c_i\text{.}\) From there, using the initial condition(s), we can determine the numerical value of each \(c_i\text{.}\) We demonstrate this approach in the following example.

Example 8.6.1.

Find a power series representation for the solution to the IVP given by
\begin{equation*} y' = y-x, \ y(0) = -1\text{.} \end{equation*}
Solution.

To begin, we assume that

\begin{equation*} y = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots\text{,} \end{equation*}

from which it follows that

\begin{equation*} y' = c_1 + 2c_2 x + 3c_3 x^2 + 4c_4 x^3 + \cdots \text{.} \end{equation*}

Setting \(y' = y - x\text{,}\) we have

\begin{align*} c_1 + 2c_2 x + 3c_3 x^2 + 4c_4 x^3 + \cdots =\mathstrut \amp (c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots) - x\\ =\mathstrut \amp c_0 + (c_1 - 1) x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots \end{align*}

Equating like coefficients, it follows that

\begin{align*} c_1 =\mathstrut \amp c_0\\ 2c_2 =\mathstrut \amp c_1 - 1\\ 3c_3 =\mathstrut \amp c_2\\ 4c_4 =\mathstrut \amp c_3\\ \vdots\mathstrut \amp \end{align*}

We can see a pattern that will continue: \(5c_5 = c_4\text{,}\) \(6c_6 = c_5\text{,}\) and so on. Further, we observe that by the given initial condition, \(y(0) = -1\text{,}\) we can find the value of \(c_0\) and this will then determine the value of the other constants. In particular, since

\begin{equation*} y = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots\text{,} \end{equation*}

it follows \(y(0) = c_0\text{,}\) so \(c_0 = -1\text{.}\) From that, we have \(c_1 = c_0 = -1\) as well. Next, since \(2c_2 = c_1 - 1\text{,}\) \(2c_2 = -2\text{,}\) so \(c_2 = -1\text{.}\) With \(3c_3 = c_2\text{,}\) we now know \(c_3 = -\frac{1}{3}\text{,}\) and using the equation involving \(c_4\) we see that \(c_4 = -\frac{1}{3 \cdot 4}\text{.}\) Similar reasoning shows that \(c_5 = -\frac{1}{3 \cdot 4 \cdot 5}\) and \(c_6 = -\frac{1}{3 \cdot 4 \cdot 5 \cdot 6}\text{.}\)

Thus, we have found the first several terms of the power series expansion for the function \(y\) that is the solution to the given IVP. In particular,

\begin{equation*} y = -1 - x - x^2 - \frac{1}{3}x^3 - \frac{1}{3 \cdot 4}x^4 - \frac{1}{3 \cdot 4 \cdot 5}x^5 - \frac{1}{3 \cdot 4 \cdot 5 \cdot 6}x^6 - \cdots \end{equation*}

The IVP in Example 8.6.1 can also be solved without using a series approach, and the solution turns out to be \(y = 1 + x - 2e^x\text{.}\) It is straightforward to verify this directly by subsituting \(y\) and \(y'\) into the differential equation to confirm that \(y' = y - x\text{,}\) as well as that \(y'(0) = -1\text{.}\) We note further that if we use the Taylor series for \(e^x\text{,}\) we also have

\begin{align*} y =\mathstrut \amp 1 + x - 2e^x\\ =\mathstrut \amp 1 + x - 2(1 + x + \frac{1}{2!}x^2 + \frac{1}{3!}x^3 + \frac{1}{4!}x^4 + \cdots)\\ =\mathstrut \amp 1 + x - 2 - 2x - x^2 - \frac{1}{3}x^3 - \frac{1}{4\cdot 3}x^4 - \cdots\\ =\mathstrut \amp -1 - x - x^2 - \frac{1}{3}x^3 - \frac{1}{4\cdot 3}x^4 - \cdots \end{align*}

which is precisely the solution we found in the example.

Subsection 8.6.2 Using power series to solve higher order initial value problems: constant coefficients

Initial value problems of the form

\begin{equation*} my'' + by' + ky = 0, \ y(0) = c_1, \ y'(0) = c_2 \end{equation*}

find important applications in physics, especially in spring-mass systems; the solution \(y\) to the IVP models a mass \(m\) that hangs from a spring with spring constant \(k\text{,}\) where the damping constant is \(b\) and the mass has an initial displacement of \(c_1\) units from equilibrium and an initial velocity of \(c_2\text{.}\) While an in-depth development of solutions to these IVPs is normally studied in a full course on differential equations, we can use power series solutions to understand some of the important basic cases.

We start with the simplest possible second-order IVP,

\begin{equation*} y'' = y, \ y(0) = 1, \ y'(0) = 1\text{.} \end{equation*}

While we know that the solution to this IVP is \(y = e^x\text{,}\) it's instructive to see how the series solution approach confirms this. In the same way that we did with first-order equations, we let

\begin{equation*} y = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots + c_n x^n + \cdots\text{,} \end{equation*}

so

\begin{equation*} y' = c_1 + 2c_2 x + 3c_3 x^2 + 4c_4 x^3 + \cdots + nc_n x^{n-1} + \cdots\text{,} \end{equation*}

and now further,

\begin{equation*} y'' = 2c_2 + 3 \cdot 2 c_3 x + 4 \cdot 3 c_4 x^2 + \cdots + n (n-1) c_n x^{n-2} + \cdots\text{.} \end{equation*}

Setting \(y'' = y\text{,}\) we see that

\begin{equation*} 2c_2 + 3 \cdot 2 c_3 x + 4 \cdot 3 c_4 x^2 + 5 \cdot 4 c_5 x^3 + \cdots = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots \text{.} \end{equation*}

Equating like coefficients,

\begin{align*} 2c_2 =\mathstrut \amp c_0\\ 3 \cdot 2 c_3 =\mathstrut \amp c_1\\ 4 \cdot 3 c_4 =\mathstrut \amp c_2\\ 5 \cdot 4 c_5 =\mathstrut \amp c_3 \end{align*}

and so on. We see from this chain of equalities that the even-indexed constants \(c_i\) depend on \(c_0\text{,}\) while the odd-indexed constants depend on \(c_1\text{.}\) In the original IVP statement, we were given that \(y(0) = 1\) and \(y'(0) = 1\text{,}\) so this implies that \(c_0 = 1\) and \(c_1 = 1\) from the power series expansions of \(y\) and \(y''\text{.}\)

Using all of our work so far, we now see that \(2c_2 = c_0 = 1\text{,}\) so \(c_2 = \frac{1}{2}\text{;}\) next, we have that \(3 \cdot 2 c_3 = c_1 = 1\text{,}\) so \(c_3 = \frac{1}{3 \cdot 2}\text{;}\) similarly, \(4 \cdot 3 c_4 = c_2 = \frac{1}{2}\text{,}\) so \(c_4 = \frac{1}{4 \cdot 3 \cdot 2}\text{;}\) and further, \(5 \cdot 4 c_5 = c_3 = \frac{1}{3 \cdot 2}\text{,}\) so \(c_5 = \frac{1}{5 \cdot 4 \cdot 3 \cdot 2}\text{.}\) We see the pattern that \(c_k = \frac{1}{k!}\text{,}\) which of course aligns with the fact that \(y = e^x\) is the solution to the original IVP. In the next activity, we explore several other second-order IVPs using a similar approach.

In the IVPs we considered in Activity 8.6.3, the respective differential equations each only involved \(y''\) and \(y\text{.}\) The situation is slightly more complicated when the second order differential equation with constant coefficients

\begin{equation*} my'' + by' + ky = 0 \end{equation*}

has all nonzero coefficients. We demonstrate how to solve such an equation using series in the following example.

Example 8.6.2.

Use power series to solve the IVP given by
\begin{equation*} y'' - 2y' - 3y = 0, \ y(0) = 4, \ y'(0) = 0\text{.} \end{equation*}
Show at least the first 5 nonzero terms of the solution \(y\text{.}\)
Solution.
We begin in the familiar way by letting
\begin{equation*} y = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + c_4 x^4 + \cdots + c_n x^n + \cdots\text{,} \end{equation*}
so
\begin{equation*} y' = c_1 + 2c_2 x + 3c_3 x^2 + 4c_4 x^3 + \cdots + nc_n x^{n-1} + \cdots\text{,} \end{equation*}
and
\begin{equation*} y'' = 2c_2 + 3 \cdot 2 c_3 x + 4 \cdot 3 c_4 x^2 + \cdots + n (n-1) c_n x^{n-2} + \cdots\text{.} \end{equation*}
Slightly rewriting the differential equation, we know that \(y\) has to satisfy the equation \(y'' = 2y' + 3y\text{.}\) Based on this form, we observe that
\begin{equation*} 2y' = 2c_1 + 4 c_2 x + 6 c_3 x^2 + 8 c_4 x^3 + \cdots + 2nc_n x^{n-1} + \cdots \end{equation*}
and
\begin{equation*} 3y = 3c_0 + 3c_1 x + 3c_2 x^2 + 3c_3 x^3 + 3c_4 x^4 + \cdots + 3c_n x^n + \cdots\text{.} \end{equation*}
Substituting the series expressions for \(y''\text{,}\) \(2y'\text{,}\) and \(3y\) in the equation \(y'' = 2y' + 3y\text{,}\) we see that
\begin{align*} 2c_2 + 6 c_3 x + 12 c_4 x^2 + 20 c_5 x^3 + \cdots =\mathstrut \amp (2c_1 + 4 c_2 x + 6 c_3 x^2 + 8 c_4 x^3 + \cdots)\\ +\mathstrut \amp (3c_0 + 3c_1 x + 3c_2 x^2 + 3c_3 x^3 + \cdots) \end{align*}
From this last equation, we can determine the relationships that exist among the constants \(c_i\text{.}\) Equating the constant terms on boths sides of the equation, we have
\begin{equation*} 2c_2 = 2c_1 + 3c_0\text{.} \end{equation*}
From the linear terms, we see
\begin{equation*} 6c_3 = 4c_2 + 3c_1\text{,} \end{equation*}
and the quadratic terms imply
\begin{equation*} 12c_4 = 6c_3 + 3c_2\text{,} \end{equation*}
and further from the cubic terms that
\begin{equation*} 20c_5 = 8c_4 + 3c_3\text{.} \end{equation*}
If we solve these equations for \(c_2\text{,}\) \(c_3\text{,}\) and so on, we have
\begin{gather*} c_2 = \frac{2c_1 + 3c_0}{2}\\ c_3 = \frac{4c_2 + 3c_1}{6}\\ c_4 = \frac{6c_3 + 3c_2}{12}\\ c_5 = \frac{8c_4 + 3c_3}{20} \end{gather*}
Using the initial conditions that \(y(0) = 4\) and \(y'(0) = 0\text{,}\) it follows from the series expansions for \(y\) and \(y'\) that \(c_0 = 4\) and \(c_1 = 0\text{,}\) so
\begin{gather*} c_2 = \frac{2 \cdot 0 + 3 \cdot 4}{2} = 6\\ c_3 = \frac{4 \cdot 6 + 3 \cdot 0}{6} = 4\\ c_4 = \frac{6 \cdot 4 + 3 \cdot 6}{12} = \frac{7}{2}\\ c_5 = \frac{8 \cdot \frac{7}{2} + 3 \cdot 4}{20} = 2\text{.} \end{gather*}
Thus, the solution to the IVP is
\begin{equation*} y = 4 + 6x^2 + 4x^3 + \frac{7}{2}x^4 + 2x^5 + \cdots\text{.} \end{equation*}

Subsection 8.6.3 Solving initial value problems with non-constant coeffiecients

For differential equations such as \(y' - 4y = 0\) or \(y'' - 2y' - 3y = 0\) that we considered in the previous section, there are well-established methods for solving these equations that don't involve using power series. But for other important differential equations, such as Airy's equation that arises in the theory of rainbows, non-constant coefficients pose a different challenge and require us to use power series in order to find a solution. Airy's equation is given by

\begin{equation*} y'' - xy = 0\text{,} \end{equation*}

and the presence of the “\(x\)” that multiplies the unknown function \(y\) makes this equation harder to solve than one like \(y'' - 4y = 0\text{.}\) Power series enable us to handle this challenge when other approaches studied in a course on differential equations fail. We explore a similar first-order equation in the following activity, and then move on to solve Airy's equation, given certain initial conditions.
